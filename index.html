Spectrometer + Fiber + Software Packages

SpectTek Co. provides integrated spectrometer packages including an Ocean Optics USB4000 spectrograph, fiber-optic connection, wavelength/response calibration, and dedicated software tools for spectral acquisition, gas analysis, and vibration-induced optical signal monitoring.

Package 1 — Gas Spectral Analysis Package

  • Ocean Optics USB4000 Miniature Fiber Optic Spectrometer
  • Approx. 196–910 nm wavelength range
  • USB 2.0 connection
  • SMA 905 optical input
  • Fiber optic measurement configuration
  • Software licence for spectra acquisition and gas processing
  • Calibration of wavelength range and optical response
This package is intended to help users detect, inspect, and measure gas spectral features associated with water vapor / water-related chemistry and carbon dioxide / carbon-bearing gas chemistry.

Package 2 — Vibration Optical Monitoring Package

  • Ocean Optics USB4000 Miniature Fiber Optic Spectrometer
  • Approx. 196–910 nm wavelength range
  • USB 2.0 connection
  • SMA 905 optical input
  • Software licence for vibration-induced optical signal analysis
  • Real-time spectrometer GUI for optical vibration monitoring
  • Calibration of wavelength range and optical response
A real-time spectrometer GUI for monitoring vibration-induced optical signals on solid metal surfaces. Designed for feasibility studies in non-contact defect monitoring using Ocean Optics / SeaBreeze spectrometers.
